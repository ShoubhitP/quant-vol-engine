# Quant Vol Engine

A quantitative options pricing and volatility modeling engine built from scratch.

## Components
- Black-Scholes pricing model
- Monte Carlo simulation engine
- Greeks calculator
- Implied volatility surface
- Heston stochastic volatility model
- GARCH volatility forecasting
- ML volatility prediction

## Tech Stack
Python, C++, NumPy, pandas, scipy, matplotlib, plotly
